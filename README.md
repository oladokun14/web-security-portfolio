# Web Security Portfolio

Hands-on cybersecurity labs completed from PortSwigger Web Security Academy.

## Overview
This repository documents my practical learning in web application security, focusing on identifying, exploiting, and understanding vulnerabilities in a controlled environment.

## Topics Covered
- Path Traversal
- Broken Access Control
- username enumeration and brute force attack

## Methodology
Each lab follows a structured approach:

1. Objective – What the lab is testing
2. Observation – What I noticed in the application
3. Exploitation – How the vulnerability was triggered
4. Impact – What an attacker could achieve
5. Mitigation – How to fix the issue
6. Key Learning – What I learned

## Tools Used
- Burp Suite
- Browser Developer Tools

## Status
Continuously updated as I progress through PortSwigger labs.

## Goal
To build practical skills in web security and document real exploitation scenarios professionally.
